# Haier Evo WM

Расширенная версия интеграции **Haier Evo** для Home Assistant с дополнительной поддержкой стиральных машин.

## О проекте

За основу взята оригинальная интеграция **haier_evo** от **and7ey**:

https://github.com/and7ey/haier_evo

Данный проект не является самостоятельной разработкой интеграции с нуля. Основная архитектура, работа с API и WebSocket принадлежат автору оригинального проекта.

В этой версии основной упор сделан на поддержку стиральных машин Haier Evo. Большинство изменений было выполнено с использованием ИИ и большого количества практического тестирования на реальном устройстве. Моих глубоких знаний протокола Haier или разработки подобных интеграций нет, поэтому любые замечания, предложения и помощь приветствуются.

Это не просто Beta, это pre-alpha 😊

## Что добавлено

На текущий момент реализовано:

* поддержка стиральных машин Haier Evo;
* определение выбранной программы;
* определение выполняемой программы;
* отображение этапов стирки;
* сенсоры мощности и потреблённой энергии;
* отображение температуры;
* отображение скорости отжима;
* отображение оставшегося времени программы;
* отображение количества оставшихся полосканий;
* сенсор прогресса программы;
* дополнительные диагностические сенсоры;
* улучшена работа WebSocket;
* исправлены некоторые проблемы с отправкой команд.

## Проверенная модель

Интеграция протестирована на:

* **Haier HW70-BP12337U1**

Работа с другими моделями возможна, но требует проверки.
